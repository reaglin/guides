Site Characteristics and Navigation

Navigation, Worksheets, and Site & Catchment Data Input in BMPFast

Prepared by: Marty Wanielista  ·  Environmental Research & Design, Inc. / Daytona State College / University of Central Florida

Section 1

Learning Objectives

Upon completion of this module, participants will be able to:

  1. Navigate the BMPFast software interface, worksheets, and button controls.
  2. Enter and interpret rainfall data, including meteorological zone selection and mean annual rainfall input.
  3. Understand and apply performance standards to surface discharge analysis.
  4. Enter and review catchment characteristics including land use, curve numbers, DCIA, and pollutant loading data.
  5. Identify the function of data fields, drop-down menus, and library support tools within BMPFast.
Design Guidance

BMPFast is designed to be easy to use by preserving information standards already accepted by the engineering profession. The software is flexible enough to respond to varying conditions across Florida's Water Management Districts (WMDs) and FDEP, and draws on the regulatory guidance contained in the Applicant's Handbooks (A.H.s).

Section 2

Background and Purpose

BMPFast was developed to address four fundamental needs in Florida stormwater management practice:

1. Consistency in ERP Assessment

There is a recognized need for consistency in evaluating stormwater management plans submitted as part of Environmental Resource Permit (ERP) applications. Results generated by BMPFast may also support flood control permits and consumptive water use permit reviews.

2. Credit for Creative and Low-Impact Solutions

The software is designed to incorporate and grant appropriate credit for creative stormwater designs. Examples of recognized innovative practices include green roofs, depression storage (rain gardens), pervious pavements, filters, swales, harvesting systems, exfiltration trenches, vegetated filter strips (VFS), tree wells, and others.

3. Florida-Based Applications

The analysis engine is grounded in Florida-specific rainfall patterns, soils, and land use conditions — not adapted from programs designed for different climates or regulatory environments. Effectiveness is evaluated on an annual basis, consistent with Florida regulatory expectations.

4. Professional Discovery and Better Design

The tool supports professional understanding of hydrologic events and encourages improved design outcomes. For example, the software facilitates analysis of disconnected impervious areas (DCIA), enabling engineers to demonstrate the water quality benefit of routing runoff away from directly connected impervious surfaces.

Section 3

Software Selection Criteria

BMPFast was selected and developed to meet all of the following evaluation criteria for Florida stormwater BMP analysis software:

  • Uses Florida rainfall data as the basis for hydrologic analysis.
  • Incorporates Florida soils and physical land use conditions.
  • Uses event mean concentration (EMC) data from Florida-based research studies.
  • Incorporates removal effectiveness values for structural BMPs, Green Stormwater Infrastructure (GSI), and Low-Impact Development (LID) practices.
  • Evaluates the combined effectiveness of BMPs arranged in series and in parallel.
  • Has been peer reviewed by practicing professionals.
  • Provides guidance to support timely completion of ERP applications.
  • Is accepted by a broad cross-section of practicing professionals.
  • Is accepted by local, regional (WMD), and FDEP regulatory and review staff.

Section 5

Navigation and Structure of the Software

BMPFast is organized around a series of worksheets that guide the user through data entry and report generation. The interface uses several standard control types to manage input.

🗂️
Figure Reference — Fig. 2
BMPFast General Site Information Worksheet — annotated diagram showing navigation buttons, data cells, drop-down menus, and unlimited entry fields. See Figure References.

Interface Control Types

Worksheets

The primary interface for entering data and retrieving results. Each worksheet corresponds to a specific phase of analysis (site info, catchment, treatment, configuration, reporting).

Navigation Buttons

Numbered buttons (1–6) on the main screen allow direct access to each workflow phase: Enter Catchment, Enter Treatment, Configure Catchments, Summary Treatment Report, Complete Report, and Cost Comparisons.

Data Cells (Unlimited Entries)

Free-entry text or numeric fields where the user types values directly. These fields have no fixed list of options and accept any appropriate input.

Drop-Down Menus (Limited Choices)

Fields that present a restricted list of pre-defined options. Used for meteorological zone selection, performance standards, groundwater analysis options, and similar constrained parameters.

The First Worksheet: Site Characteristics

The General Site Information worksheet is the entry point for every new BMPFast project. Data entered here establishes the foundational parameters that govern all subsequent catchment and treatment calculations. The user begins data entry in this worksheet immediately after the welcome screen.

Key Features of the Site Characteristics Worksheet

  • Dynamic input: Any field value can be changed at any time without reloading or restarting the project. The software recalculates results based on the updated inputs.

Section 7

General Site Information Fields Explained

Project Name

A free-text identifier for the project file. The project name is displayed in the title bar of every worksheet and appears in all generated reports.

Meteorological Zone

Florida is divided into five designated meteorological zones for stormwater analysis purposes. The user selects the zone that corresponds to the project location from a drop-down menu. A library button opens the Meteorological Zones and Mean Annual Rainfall Map, which displays the entire state or can be filtered by region (Northwest, Northeast, Central, South) and includes access to recent data. Zone selection determines the rainfall frequency distributions and storm event data applied in the analysis.

Mean Annual Rainfall

The user enters the site-specific mean annual rainfall in inches. This value is obtained from the zone map and recent NOAA data. The software includes the 2024 updated rainfall isopleth map (based on NOAA 2022 data) for reference. The software provides an open entry field to accommodate locations where updated values differ from zone defaults or where future revisions are anticipated.

Performance Standard of Surface Discharge Analysis

This drop-down field defines the regulatory performance standard against which the stormwater management system will be evaluated. The selection determines the required removal efficiencies for Total Phosphorus (TP) and Total Nitrogen (TN), and sets any additional criteria such as post-development loading limits or water quality targets in receiving waters.

Conduct Groundwater Discharge Analysis

When set to "Yes," BMPFast activates an additional analysis pathway for groundwater-influenced systems. Groundwater pollutant loadings are calculated and added to the surface water loadings for a combined analysis. This setting is required for BMP systems such as exfiltration trenches, retention ponds with groundwater interaction, and rain gardens in areas with shallow water tables.

Nitrogen and Phosphorus Removal Efficiency Display

Once the performance standard is selected, the worksheet displays the applicable Nitrogen Removal Efficiency (%) and Phosphorus Removal Efficiency (%) targets. These values are auto-populated from the regulatory table and update when the performance standard selection is changed. For example, "All sites non-exempted" yields 55% TN removal and 80% TP removal.

Section 8

Performance Standards

The performance standard selection is one of the most consequential inputs in the General Site Information worksheet. It determines the required removal targets for both Total Phosphorus and Total Nitrogen across the entire project. The table below summarizes the available performance standard options and their associated criteria.

Project Scenario TP Removal (%) TN Removal (%) Additional Criteria
All Sites (non-exempted) 80 55 or Post ≤ Pre
OFW 90 80 or Post ≤ Pre
Impaired Water 80 80 & Post ≤ Pre + WQ Targets
Impaired Water + OFW 95 95 & Post ≤ Pre + WQ Targets
Redevelopment 80 45 none
Redevelopment + OFW 90 60 none
Redevelopment + Impaired Water 80 45 and WQ Targets
Specified Removal Efficiency User-defined % —
Net Improvement Post < Pre —
BMP Analysis Analytical only —
Regulatory Note

WQ Targets are the water quality standards applicable in the receiving water body (net improvements above pre-development baseline). OFW = Outstanding Florida Waters. These scenarios address BMAP (Basin Management Action Plan), TMDL (Total Maximum Daily Load), and other local program requirements.

Additional Standards

The "Net Improvement," "Specified Removal Efficiency," and "BMP Analysis" options provide flexibility to address BMAP, TMDL, and other local program needs that fall outside the standard ERP performance tiers. These options allow the engineer to define custom removal targets or perform comparative analyses without regulatory percentage thresholds.

Section 10

Recent Rainfall Data in BMPFast: 2024 Annual Rainfall

BMPFast 2025 includes an updated annual rainfall isopleth map based on NOAA 2022 data, published in 2024. This map is accessible via the "Recent Data" button in the meteorological zone library view. Users should consult this map when entering the Mean Annual Rainfall to ensure their project uses current, site-appropriate values.

🌧️
Figure Reference — Fig. 5
Average Annual Rainfall Isopleth Map for Florida (NOAA 2022 / Published 2024) — showing rainfall contours ranging from approximately 40 inches in some northern areas to 66 inches in portions of south Florida. See Figure References.

The isopleth map displays contour lines of equal annual rainfall depth across the state. The legend ranges from 40 to 66 inches, with the color gradient moving from warm tones (lower rainfall in north Florida) to cool blue-green tones (higher rainfall in south and southwest Florida). Users should identify the contour interval that encompasses their project site and enter the corresponding depth in the Mean Annual Rainfall field.

Some annual rainfalls under the updated data are greater, some are smaller, and some are approximately equal to the 2000 values. Engineers must consult the current 2024 map for accurate design inputs. The software includes an open entry field specifically to accommodate updated or locally-derived rainfall values, recognizing that continued revisions to this map are expected.

Section 12

Entering Catchment Data

After completing the General Site Information worksheet, the user navigates to the Watershed Characteristics Worksheet by clicking the "1. Enter Catchment" button. This worksheet captures the physical and water quality characteristics of each drainage catchment contributing to the stormwater management system.

🏞️
Figure Reference — Fig. 7
Watershed Characteristics Worksheet — showing catchment name entry, land use selection (pre and post-development), catchment area fields, curve number entries, DCIA percentage, EMC values, Annual ROC, runoff volumes, and pollutant loading calculations. See Figure References.

Catchment Identification

The worksheet displays the current catchment number and allows the user to assign a descriptive name (e.g., "Venice LID Award Road"). Multiple catchments can be added using the "Add Catchment" button, and each is tracked as a separate tab within the worksheet.

Land Use Classification

Pre-development and post-development land use categories are selected from drop-down menus. These selections determine the Event Mean Concentrations (EMC) for Total Nitrogen and Total Phosphorus used in the loading calculations. The user may also select "User Defined Values" to enter site-specific EMC data.

Groundwater Note

When Groundwater Discharge Analysis is set to "Yes" in the Site Characteristics worksheet, the catchment worksheet includes an additional calculation step for groundwater pollutant loadings. These groundwater loadings are summed with the surface water loadings before reporting. After completing the groundwater analysis within the catchment, the user must click the Calculate button, followed by the Back button to return to the main worksheet.

Section 13

Project Workflow: From Site Info to Reports

The complete BMPFast project workflow follows a sequential path through the six main navigation steps. The sequence must be followed in order, as later steps depend on data entered in earlier ones.

General Site Information

Enter project name, select meteorological zone, input mean annual rainfall, select performance standard, and specify whether to conduct a groundwater discharge analysis. Set baseline nitrogen and phosphorus removal efficiency targets.

Enter Catchment Data

For each catchment, enter land use classifications (pre and post), catchment areas, non-DCIA curve numbers, and DCIA percentages. Review calculated EMC values, runoff volumes, and pollutant loadings. Add additional catchments as needed.

Enter Treatment Data

Select and configure BMP treatment systems (retention ponds, filters, swales, LID practices, etc.). Define the design parameters for each BMP.

Configure Catchments

Assign catchments to treatment systems and define the routing relationships between catchments and BMPs (series, parallel, or combined configurations).

Summary Treatment Report

Review the calculated treatment performance summary. Verify that the configured system meets the required performance standard for TP and TN removal.

Complete Report & Cost Comparisons

Generate a complete project report for ERP submission. Optionally run cost comparisons between alternative BMP configurations. Save the project file for future reference and reuse.

Training Version Limitation

In the training version of BMPFast, the Summary Report (Step 4), Save Project, and Open Project functions are disabled. Users completing training exercises will follow Steps 1 through 3 (and Step 6 for Cost Comparisons) to develop competency. Training is highly recommended before use in production ERP work. Basic competency in BMPFast helps reduce RAIs (Requests for Additional Information) and facilitates timely permit approvals.

Module Summary

Learning Objectives Review

Upon completion of this module, participants should be able to accomplish the following:

Module 5 Competencies

  1. Navigation of Software: Locate and use all worksheets, navigation buttons, data cells, drop-down menus, and library support tools within BMPFast.
  2. Rainfall Data Input: Select the appropriate meteorological zone, look up current annual rainfall from the 2024 isopleth map, and enter the value in the Mean Annual Rainfall field.
  3. Performance Standards: Identify and select the correct performance standard (All Sites, OFW, Impaired Water, Redevelopment, Net Improvement, etc.) based on project conditions and receiving water classification.
  4. Catchment Characteristics: Enter pre- and post-development land use, catchment areas, SCS curve numbers, and DCIA percentages; interpret calculated EMC values, annual runoff coefficients, and pollutant loading outputs.
  5. Data Fields and Library: Use the library/support buttons to access zone maps, rainfall tables, and reference data without leaving the worksheet; understand the distinction between unlimited data cells and limited drop-down menus.
