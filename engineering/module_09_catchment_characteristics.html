Catchment Characteristics

By Marty Wanielista  ·  January 2026

Learning Objectives

After completing this module, you will be able to:

  1. Define what a catchment is and how it is delineated.
  2. Identify and use worksheet inputs and outputs in BMPFast.
  3. Locate relevant EMC values in the Applicant's Handbook.
  4. Apply user-defined Event Mean Concentration (EMC) values.
  5. Combine multiple land uses within a single catchment.
  6. Account for wet pond area in catchment loading calculations.

Definitions

Best Management Practice (BMP)

Source: Applicant's Handbook, Volume 1, Section 2 — Definitions and Terms, June 28, 2024

A Best Management Practice (BMP) for stormwater treatment means a practice or combination of practices, based on research, field-testing, and expert review, to be the most effective and practicable — including economic and technological considerations — for improving water quality by reducing excess nutrients and other pollutant loads in water.

Traditional BMPs for stormwater treatment are identified and described in Appendix O and the applicable Applicant's Handbook Volume II. Provisions for proposing alternative BMPs are specified in Section 9.5.2 of Volume I.

Catchment

A catchment is an area which discharges runoff water to a surface water or groundwater through retention. Within or at the discharge point, there is potential for the management of stormwater by a BMP.

The catchment area is defined by a point of discharge and the possible BMP location at or near that point.

Catchment Configuration and Flow Direction

A catchment is delineated based on topography and drainage patterns that converge at a single discharge point. Understanding flow direction and catchment boundaries is essential before entering data into BMPFast.

A single project site may contain multiple catchments, each with its own discharge point and potential BMP location. The flow relationships between catchments — which drain to which — must be established before analysis begins.

The Watershed Characteristics Worksheet is used to define each catchment's pre- and post-development land use, physical characteristics, and pollutant loadings. A catchment name and land use must be entered along with all corresponding data fields. Once the first catchment is fully entered, additional catchments can be added using the Add Catchment button.

Note on DCIA

Directly Connected Impervious Area (DCIA) is the percentage of the catchment area consisting of impervious surfaces that drain directly to the stormwater collection system without first passing over pervious areas. DCIA is entered as a percentage (0–100%). Surfaces such as rooftops draining to grass before reaching the collection system are NOT considered DCIA.

Multiple Land Uses in a Catchment

When a catchment contains more than one land use type, a composite Curve Number (CN) must be calculated. BMPFast provides a Composite CN Calculator for this purpose. The user enters the area (in acres) and CN value for each land use sub-area. For impervious areas, CN = 95 is used by convention.

The key distinction is that BMPFast does not use the area-weighted average CN directly. Instead, the software adds the individual runoff volumes calculated for each land use area and then back-calculates the single equivalent CN that produces the same total runoff. This approach more accurately represents the non-linear relationship between CN and runoff depth.

Example: 12-Acre Single-Family Development

Area (ac) CN C Weighted C
8.219 50 0.230 0.216
0.541 95 0.513 0.032

Avg Weighted C: 0.247

DCIA Percent: 27.0%

Area-Averaged CN: 52.78

Simple Average CN: 72.5

CN used in BMPFast (back-calculated from runoff volumes):

64.00

Why the Back-Calculated CN Differs

The simple arithmetic average of CN values (72.5 in this example) overestimates runoff because it ignores the area weighting. The area-weighted average (52.78) is closer but still not used directly. BMPFast sums the runoff from each land-use area individually, then solves for the equivalent CN — yielding CN = 64 in this example. This value is what should be entered into the catchment worksheet.

Wet Pond Contribution to Loadings

A wet pond (permanent pool) within the catchment boundary is treated differently from other land areas in BMPFast. The water surface of a wet pond does not generate runoff-borne pollutant loads in the same manner as upland land uses. Users enter the wet pond surface area (in acres) in the designated field on the catchment worksheet, and BMPFast excludes this area from land-use-based loading calculations.

The example below compares calculated outputs for the same 12-acre single-family catchment with and without a one-acre wet pond surface area entered.

Output Parameter Without Wet Pond (0 ac) With 1-Acre Wet Pond
Annual Runoff Coefficient (ROC) 0.022 0.260
Runoff (ac-ft/yr) 1.144 12.374
N Loading (kg/yr) 2.976 27.006
P Loading (kg/yr) 0.254 4.989
Interpretation

Adding a one-acre wet pond surface changes the catchment loading outputs significantly because the open water area has its own rainfall-based loading characteristics distinct from the upland land use. Accurately entering wet pond area ensures the loading analysis correctly apportions nutrient contributions from each land cover type.

User Defined EMC Values

By default, BMPFast assigns Event Mean Concentration (EMC) values for nitrogen and phosphorus based on the selected land use type from the Applicant's Handbook lookup tables. However, the software provides a User Defined Values option in the Post-Development land use dropdown, allowing engineers to input site-specific or agreed-upon EMC values when standard handbook values are not appropriate.

User-defined EMC values are commonly used when a portion of the Directly Connected Impervious Area (DCIA) has been physically disconnected from the storm system — for example, rooftop drainage redirected to infiltration areas. In these cases, separate EMC values are negotiated with the permitting agency for the impervious and pervious sub-areas.

Composite EMC Calculation

When user-defined values are selected, BMPFast opens the Composite EMC Calculator, which functions similarly to the Composite CN Calculator. The user enters the area, EMC, and CN for each sub-area, and the tool computes the weighted composite EMC.

Module Summary — What Did We Learn?

  1. What a catchment is — an area defined by a single discharge point where BMP placement is possible.
  2. Worksheet inputs and outputs — land use, area, CN, DCIA percentage, wet pond area, and non-runoff pollutant loads; outputs include annual runoff coefficient, runoff volume, and N and P loadings. Noted how EMC values change between pre- and post-development land use selections.
  3. User Defined EMCs — how to enter site-specific or agency-agreed EMC values when standard handbook values are not applicable, including partial DCIA disconnection scenarios.
  4. Combined land uses and CN/EMC calculations — use of the Composite CN Calculator to compute a single equivalent curve number from multiple sub-areas, and the analogous Composite EMC Calculator for weighted average pollutant concentrations.
  5. Wet pond area — how entering the wet pond surface area in the catchment worksheet affects loading calculations by segregating the open-water area from upland land use loading.
